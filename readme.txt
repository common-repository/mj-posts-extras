Plugin for displaying various lists of posts, comments, etc.

== Description ==

This plugin is intended to display anything related to posts, or single post pages. Current release (first) has abilities to display most commented (popular) posts, posts from same category, posts with same custom post type and latest comments from all posts. Future releases will add more display options.

== Installation ==

1. Download plugin from http://markojakic.net/mjpx-download/mjpx_posts_extras.zip
2. Unpack
4. Upload 'mjpx_posts_extras' directory to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
